## Цикли та функції в Python

**Мета:** навчитися використовувати цикли та функції для автоматизації операцій та полегшення коду.

---

## 1. Теоретичні відомості

### Цикли

**`for`** — обхід послідовностей:
```python
for i in range(1, 6):       # числа 1..5
    print(i)

for item in ["a", "b", "c"]:  # список
    print(item)
```

**`while`** — виконується поки умова True:
```python
n = 0
while n < 5:
    n += 1
```

**`break`** — зупиняє цикл, **`continue`** — пропускає поточну ітерацію.

---

### Функції

```python
def greet(name, greeting="Привіт"):
    return f"{greeting}, {name}!"

greet("Аня")            # Привіт, Аня!
greet("Аня", "Добрий день")
```

**`*args`** — довільна кількість позиційних аргументів.  
**`**kwargs`** — довільна кількість іменованих аргументів.  
**`lambda`** — анонімна однорядкова функція.

---

### Вищий порядок та рекурсія

```python
# map, filter, reduce
list(map(lambda x: x**2, [1,2,3]))     # [1, 4, 9]
list(filter(lambda x: x>2, [1,2,3]))   # [3]

# Рекурсія
def factorial(n):
    return 1 if n <= 1 else n * factorial(n-1)
```

---

### Бібліотеки

**`math`** — `sqrt`, `pi`, `factorial`, `log`, `ceil`, `floor`.  
**`itertools`** — `chain`, `cycle`, `combinations`, `permutations`, `accumulate`, `groupby`.

## 3. Висновки

- **`for`** зручний для обходу відомої послідовності, **`while`** — коли кількість ітерацій наперед невідома.
- **Функції** дозволяють виділити повторюваний код у окремий блок, що спрощує читання та тестування.
- **Рекурсія** елегантно вирішує задачі на кшталт факторіалу чи Фібоначчі, але потребує базового випадку.
- **`lambda`, `map`, `filter`** роблять код лаконічним при трансформації та фільтрації даних.
- **Замикання та декоратори** — потужні інструменти для розширення поведінки функцій без зміни їх коду.
- **`itertools`** та **`math`** значно розширюють можливості без написання зайвого коду вручну.
- Поєднання циклів і функцій — основа будь-якої реальної програми: від сортування до аналізу тексту.
