## 1. Теоретичні відомості

### Множина (set)

Множина — невпорядкована колекція **унікальних** елементів. Не підтримує індексацію. Ідеальна для перевірки належності та операцій над групами даних.

```python
s = {1, 2, 3, 4, 5}
s.add(6)          # додати елемент
s.discard(1)      # видалити (без помилки якщо немає)
s.remove(2)       # видалити (помилка якщо немає)
```

**Операції над множинами:**

| Операція | Символ | Метод |
|---|---|---|
| Об'єднання | `a \| b` | `a.union(b)` |
| Перетин | `a & b` | `a.intersection(b)` |
| Різниця | `a - b` | `a.difference(b)` |
| Симетрична різниця | `a ^ b` | `a.symmetric_difference(b)` |

**`frozenset`** — незмінна версія множини, може бути ключем словника.

---

### Словник (dict)

Словник — колекція пар **ключ → значення**. Ключі унікальні та незмінні. Починаючи з Python 3.7 зберігає порядок вставки.

```python
d = {"name": "Аня", "age": 21}
d["city"] = "Київ"          # додати / змінити
d.pop("age")                # видалити за ключем
val = d.get("email", "N/A") # безпечний доступ
```

**Основні методи:** `keys()`, `values()`, `items()`, `update()`, `setdefault()`, `pop()`, `popitem()`, `copy()`, `clear()`.

---

### Словникові включення

```python
squares  = {x: x**2 for x in range(1, 6)}
filtered = {k: v for k, v in d.items() if v > 10}
```

---

### Часова складність

| Операція | list | set / dict |
|---|---|---|
| Пошук елемента | O(n) | **O(1)** |
| Вставка | O(1) | O(1) |
| Видалення | O(n) | O(1) |

Set та dict значно швидші за list при пошуку завдяки хеш-таблиці.

## 3. Висновки

У процесі виконання практичної роботи було освоєно:

- **Множина** гарантує унікальність елементів автоматично та дозволяє виконувати математичні операції (об'єднання, перетин, різниця) над групами даних одним рядком коду.
- **Словник** є основним інструментом для зберігання структурованих даних з іменованим доступом. Завдяки хеш-таблиці пошук виконується за O(1) незалежно від розміру.
- **Конфлікти ключів** при злитті словників вирішуються через `update()` або `{**a, **b}` — порядок аргументів визначає, чиє значення переможе.
- **`setdefault()`** зручний для групування даних: дозволяє ініціалізувати ключ тільки якщо його ще немає.
- **Словникові включення** роблять код компактним та читабельним при фільтрації та трансформації даних.
- У **реальній задачі** (інвертований індекс) поєднання словника та множин дозволило реалізувати пошук за ключовими словами через перетин множин — ефективно та лаконічно.

Множини та словники є незамінними структурами в Python для задач агрегації, дедублікації, індексування та швидкого пошуку.
